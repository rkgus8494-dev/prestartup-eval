예비창업패키지 모의 심사 시뮬레이터

심사위원은 “근거”를 봅니다. (없음/부분/충족 3단계)
“부분”은 근거가 일부는 있지만 제출 기준에 맞게 보완이 필요한 상태입니다.

진단 입력

각 문항은 “증빙 가능 여부” 기준으로 선택하세요. (없음=0 / 부분=0.5 / 충족=1)

사용법

  1. 각 항목을 ‘없음/부분/충족’으로 체크
  2. ‘부분(0.5)’을 선택하면 빠진 요소(수치/증빙/구체화)를 체크
  3. 맨 아래 결과 산출을 누르면 결과 페이지로 이동
  4. 결과 페이지에서 PDF 저장 가능

가점 항목 (최대 +10)

공고별 가점은 다릅니다. 해당되면 체크하세요.
* “부분”이 많은 상태로 제출하면 감점 위험이 큽니다.

요약 결과

점수/리스크/보완 문장 제안을 출력합니다.
아직 결과가 없습니다.

탈락 위험 TOP3

미충족/부분 항목에서 영향도가 큰 3개를 뽑습니다.

보완 문장 자동 제안

TOP3 + ‘부분에서 체크한 빠진 요소’까지 반영해 자동 제안합니다.
